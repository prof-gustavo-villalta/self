## Envio oficial

Use o formulário informado pelo professor em sala para registrar a atividade da
aula.

Enquanto o link oficial não estiver publicado, organize as evidências abaixo no
seu repositório ou em uma pasta compartilhável.

---

## O que você deve informar

| Ordem | Pergunta                                                                 | Tipo de resposta |
| :---- | :----------------------------------------------------------------------- | :--------------- |
| 1     | Nome completo                                                            | Resposta curta   |
| 2     | Qual endpoint foi usado na atividade?                                    | Resposta curta   |
| 3     | Para que serve o método `fromJson`?                                      | Parágrafo        |
| 4     | Qual pacote Flutter foi usado para fazer requisições HTTP?               | Múltipla escolha |
| 5     | Em qual arquivo Android fica a permissão de internet?                    | Múltipla escolha |
| 6     | Link para o repositório no GitHub                                        | Resposta curta   |
| 7     | Link para evidências da tela carregando, lista funcionando e tela de erro | Resposta curta   |
| 8     | Qual foi sua maior dificuldade na aula?                                  | Parágrafo        |

---

## Antes de abrir o formulário

- confirme se a API abre no navegador;
- confirme se o app mostra a lista de usuários;
- confirme se existe uma tela de erro ou mensagem amigável;
- registre pelo menos 3 evidências visuais;
- verifique se o repositório está atualizado.
