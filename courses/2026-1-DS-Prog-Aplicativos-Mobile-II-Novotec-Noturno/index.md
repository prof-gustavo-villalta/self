## Ementa

### Competências

1. Desenvolver aplicativos mobile multiplataforma com Flutter
2. Criar interfaces responsivas e intuitivas
3. Consumir APIs REST e implementar comunicação em tempo real
4. Integrar recursos nativos do dispositivo (câmera, sensores, GPS)
5. Implementar sistemas de autenticação e segurança de dados
6. Publicar aplicativos nas lojas oficiais (Google Play/App Store)

### Conteúdos Programáticos

#### Unidade 1: Conectividade

- Consumo de APIs REST
- Comunicação TCP full-duplex (sockets)
- Integração com dispositivos embarcados via Bluetooth

#### Unidade 2: Autenticação

- Mecanismos de autenticação em aplicativos mobile
- Segurança de dados do usuário
- Armazenamento seguro de credenciais

#### Unidade 3: Recursos do Dispositivo

- Câmera e galeria de imagens
- Sensores (acelerômetro, giroscópio, etc.)
- Localização, orientação e mapas (GPS)
- Telefonia e SMS

#### Unidade 4: Empacotamento e Distribuição

- Geração de APK/IPA
- Assinatura de aplicativos
- Publicação nas lojas (Google Play/App Store)
- Configurações de build e release
