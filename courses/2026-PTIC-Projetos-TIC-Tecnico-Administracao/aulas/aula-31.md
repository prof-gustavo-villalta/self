# Aula 31 - Sistematização

**Data:** 30/06/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Revisar os principais conteúdos do primeiro semestre e preparar a
estrutura final do portfólio digital.

---

## 1. Abertura: O Que Construímos? (10 minutos)

### Olhando para Trás

Desde a primeira aula de organização de pastas até os algoritmos complexos,
percorremos um longo caminho. Hoje vamos consolidar esse conhecimento para que
ele não se perca e para que o portfólio reflita essa evolução.

### Guia para o Professor

- **Instruções Didáticas:** Projete o cronograma do semestre. Relembre cada
  etapa: Nuvem -> Segurança -> Office -> Lógica.
- **Perguntas para a Turma:**
  - "Qual dessas competências você acha que usará mais em um estágio?"
  - "Qual arquivo do seu computador hoje está mais organizado do que em março?"
- **Dicas de Condução:** Peça para os alunos compartilharem uma "vitória"
  técnica que tiveram no semestre.

---

## 2. Teoria: Portfólio como Evidência (15 minutos)

### A Vitrine do Profissional

O portfólio não é apenas uma pasta de arquivos. É a prova de que você sabe
fazer.

- **Organização:** Pastas nomeadas por aula ou competência.
- **Qualidade:** Arquivos finais limpos, sem versões "teste1", "copia_final".
- **Reflexão:** Um pequeno texto explicando o que cada atividade ensinou.

### Guia para o Professor

- **Instruções Didáticas:** Mostre um exemplo de portfólio bem organizado.
  Explique que recrutadores valorizam a capacidade de organização tanto quanto o
  conhecimento técnico.
- **Perguntas para a Turma:**
  - "Se um chefe pedisse para ver seu trabalho de hoje, você teria orgulho de
    mostrar?"
  - "O que falta no seu portfólio para ele ficar nota 10?"
- **Dicas de Condução:** Reforce a importância do backup na nuvem para não
  perder o semestre inteiro.

---

## 3. Atividade Prática

Os alunos devem dedicar este tempo para:

1. Revisar todos os arquivos criados no semestre.
2. Renomear arquivos fora do padrão (ex: `aula25_vfinal.pseint` ->
   `Aula-25-Logica-Decisao.psc`).
3. Verificar se todos os formulários Google foram respondidos.
4. Criar um documento "Índice" no portfólio listando as 5 principais habilidades
   desenvolvidas.

### Guia para o Professor

- **Instruções Didáticas:** Passe de computador em computador fazendo uma
  "auditoria rápida". Aponte desorganizações. Ajude quem perdeu arquivos a
  recuperá-los do Google Drive/Lixeira.
- **Perguntas para a Turma:**
  - "Qual atividade foi sua favorita?"
  - "Você consegue explicar seu algoritmo de logística para alguém que não sabe
    programar?"
- **Dicas de Condução:** Transforme a aula em um momento de colaboração: quem
  terminou ajuda quem está atrasado.

### Entrega

- Registrar a atividade no [Formulário da Aula 31](formulario-aula-31).
- Anexar ou informar o link do arquivo, pasta, apresentação ou evidência
  produzida.
- Manter o portfólio organizado com nome de arquivo coerente e data da aula.---

## 5. Dicas de Ouro para o Professor

1. **Checklist:** Forneça um checklist impresso ou digital com todas as
   atividades que devem estar no portfólio.
2. **Feedback Individual:** Use este tempo para dar feedbacks rápidos sobre a
   evolução de cada aluno.
3. **Motivação:** Lembre-os que o segundo semestre trará a Inteligência
   Artificial, e que a base de lógica será fundamental para isso.
