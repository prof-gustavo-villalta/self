# Aula 57 - Projeto: Modelagem

**Data:** 20/10/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Estruturar a solução digital por meio de fluxogramas e modelagem
de processos, definindo como os dados e as interações vão ocorrer.

---

## 1. Abertura: O Esqueleto da Solução (11 minutos)

### O que é Modelagem?

Antes de construir a solução no Excel ou no Canva, precisamos desenhar como ela
funciona "por dentro". Quais os passos que o usuário faz? Quais as decisões que
o sistema toma? Hoje vamos usar o **Draw.io** para desenhar o fluxograma da
nossa solução.

### Guia para o Professor

- **Instruções Didáticas:** Revise brevemente os símbolos de fluxograma que
  aprendemos no primeiro semestre (Início, Decisão, Processo, Documento, Fim).
- **Perguntas para a Turma:**
  - "Qual o primeiro passo do seu usuário quando ele abre a sua solução?"
  - "O que acontece se a resposta da IA for errada? Para onde o processo deve
    ir?"
- **Dicas de Condução:** Mostre que a modelagem serve para **antecipar erros**.

---

## 2. Teoria: Processos e Fluxos de IA (17 minutos)

### Como a IA entra no Fluxo?

Em um projeto integrador, a IA não é apenas uma caixa preta. Ela é uma etapa:

1. **Entrada:** Dados do usuário (ex: Nome do currículo).
2. **Processamento IA:** O prompt (ex: "Resuma este currículo em 3 tópicos").
3. **Validação Humana:** Alguém revisa a IA (Passo essencial!).
4. **Saída:** Documento formatado no Word ou PDF.

### Guia para o Professor

- **Instruções Didáticas:** Explique que o fluxograma ajuda a identificar onde a
  IA é necessária e onde o trabalho humano é obrigatório.
- **Perguntas para a Turma:**
  - "Onde o seu projeto tem uma tomada de decisão importante?"
- **Dicas de Condução:** Incentive a clareza visual. O fluxograma deve ser lido
  por qualquer pessoa sem explicação.

---

## 3. Atividade Prática

Os grupos devem criar o fluxograma da sua solução no Draw.io:

1. **Fluxo Principal:** O caminho feliz (quando tudo dá certo).
2. **Tratamento de Erros:** O que fazer se o dado vier errado?
3. **Integração:** Indique onde as ferramentas de Office e IA se conectam.

**Tarefa:** Exporte o fluxograma como imagem (.png ou .pdf) e salve na pasta do
grupo.

### Guia para o Professor

- **Instruções Didáticas:** Passe nos grupos auxiliando na lógica. Verifique se
  os símbolos de "Decisão" (Losango) têm sempre dois caminhos (Sim/Não).
- **Perguntas para a Turma:**
  - "Quem é o responsável por revisar os dados gerados pela IA no seu fluxo?"
- **Dicas de Condução:** Peça que eles usem cores diferentes para identificar
  tarefas manuais, tarefas de IA e tarefas de Office.

## 5. Dicas de Ouro para o Professor

1. **Visão Sistêmica:** Lembre que o fluxograma é o "Manual de Instruções" para
   a fase de Desenvolvimento (Aula 58).
2. **Simplicidade:** Se o fluxo está complexo demais, o projeto pode não ser
   entregue a tempo. Ajude a "limpar" o processo.
3. **Correção:** Dedique os últimos 5 minutos para que um grupo veja o
   fluxograma do outro e dê sugestões.
