# Aula 49 - Prática Integrada

**Data:** 21/09/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Integrar múltiplas ferramentas de IA (texto, imagem, slides e
planilhas) para criar uma campanha digital completa para uma iniciativa da
escola ou empresa fictícia.

---

## 1. Abertura: O Orquestrador de IAs (10 minutos)

### Unindo as Peças

Até agora, usamos cada IA em seu "quadrado". Hoje, vocês serão os regentes de
uma orquestra tecnológica. Vamos usar o ChatGPT para o roteiro, o Canva para as
artes, o Gamma para a apresentação e o Excel para o orçamento. O objetivo é ver
como essas ferramentas conversam entre si para criar um projeto robusto em tempo
recorde.

### Guia para o Professor

- **Instruções Didáticas:** Explique o conceito de **Workflow (Fluxo de
  Trabalho)**. Mostre como o resultado de uma IA alimenta a próxima (ex: o texto
  do chat vai para o gerador de slides).
- **Perguntas para a Turma:**
  - "Qual a maior dificuldade de fazer ferramentas diferentes trabalharem
    juntas?"
  - "Como manter a mesma 'voz' e o mesmo 'estilo' visual em todas elas?"
- **Dicas de Condução:** Use um exemplo real da escola (ex: Divulgação da Feira
  de Profissões).

---

## 2. Teoria: Consistência de Marca com IA (15 minutos)

### A Importância do Branding

Quando criamos várias peças, elas precisam parecer parte da mesma família.

1. **Consistência de Texto:** Use o mesmo prompt de "Persona" em todos os chats.
2. **Consistência Visual:** Use a mesma "Paleta de Cores" e "Estilo de Imagem"
   em todas as gerações.
3. **Fluxo de Dados:** O orçamento na planilha deve bater com os valores citados
   nos slides e no e-mail.

### Guia para o Professor

- **Instruções Didáticas:** Mostre como criar um "Guia de Estilo" simples em um
  prompt para a IA seguir em todas as tarefas da aula.
- **Perguntas para a Turma:**
  - "O que acontece se o e-mail for formal e o post do Instagram for cheio de
    gírias?"
  - "Como a IA pode ajudar a adaptar o mesmo texto para diferentes canais?"
- **Dicas de Condução:** Relembre conceitos de marketing básico aplicados à
  administração.

---

## 3. Atividade Prática

Em grupos ou individualmente, os alunos devem criar os seguintes materiais para
a campanha:

1. **Planejamento (Texto):** Um plano de ação de 5 tópicos gerado pela IA.
2. **Divulgação (Imagem):** Um cartaz digital criado com IA de imagem.
3. **Apresentação (Slides):** 3 slides rápidos para apresentar a ideia para a
   direção.
4. **Orçamento (Planilha):** Uma tabela simples com estimativa de custos para
   lixeiras de coleta seletiva.

**Requisito:** Todas as peças devem usar o mesmo nome de projeto e a mesma
identidade visual.

### Guia para o Professor

- **Instruções Didáticas:** O tempo é curto, então o foco é a integração.
  Observe se eles estão conseguindo levar as informações de uma ferramenta para
  outra sem erros.
- **Perguntas para a Turma:**
  - "Qual foi a parte mais rápida de integrar?"
  - "Onde houve mais 'retrabalho' manual?"
- **Dicas de Condução:** Circule ajudando a "conectar os fios" entre as
  ferramentas.

## 5. Dicas de Ouro para o Professor

1. **Trabalho em Equipe:** Esta aula funciona muito bem em grupos, onde cada
   aluno domina uma ferramenta de IA (o "especialista em imagem", o
   "especialista em texto", etc).
2. **Agilidade:** Reforce que o objetivo não é a perfeição, mas a
   **viabilidade** de um projeto integrado.
3. **Links de Compartilhamento:** Peça para criarem uma pasta única no Drive
   para a Campanha.
