# Aula 48 - Oficina: Portfólio IA

**Data:** 15/09/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Compilar, organizar e documentar todas as criações realizadas com
suporte de IA no semestre, criando uma vitrine de competências tecnológicas.

---

## 1. Abertura: Sua Identidade Digital 2.0 (10 minutos)

### Mais que arquivos, competências

No primeiro semestre, vocês organizaram documentos tradicionais. Agora, o
portfólio de vocês deve mostrar que vocês sabem **pilotar** as ferramentas do
futuro. Um empregador não quer saber apenas se você usou IA, mas **como** você a
usou para gerar valor.

### Guia para o Professor

- **Instruções Didáticas:** Mostre como um profissional moderno exibe seus
  trabalhos (ex: LinkedIn, GitHub, sites de portfólio). Explique que o Google
  Drive será nossa base, mas a organização deve ser impecável.
- **Perguntas para a Turma:**
  - "Quais são os 3 melhores trabalhos com IA que você fez até agora?"
  - "Como você explicaria para um chefe que você usou IA de forma ética?"
- **Dicas de Condução:** Relembre a Aula 31 (Sistematização do 1º Semestre).

---

## 2. Teoria: Documentando o Processo (15 minutos)

### Não mostre apenas o ovo, mostre o processo

Para cada trabalho de IA no portfólio, é ideal ter:

1. **O Problema Original:** O que precisava ser resolvido.
2. **O Prompt Utilizado:** Qual comando você deu para a máquina.
3. **A Resposta da IA:** O resultado bruto.
4. **Sua Intervenção Humana:** O que você mudou, corrigiu ou melhorou.
5. **O Resultado Final:** O documento pronto e revisado.

### Guia para o Professor

- **Instruções Didáticas:** Explique que o valor do aluno está na etapa 4
  (Intervenção Humana). É isso que prova que ele não foi apenas um "copiador".
- **Perguntas para a Turma:**
  - "Por que guardar o prompt é importante?" (Reprodutividade e melhoria
    contínua).
  - "Como nomear os arquivos para que fiquem fáceis de achar?"
- **Dicas de Condução:** Sugira uma estrutura de pastas:
  `Portfólio/Semestre2/IA_Generativa/Textos`, `/Imagens`, `/Slides`.

---

## 3. Atividade Prática

Os alunos devem dedicar este tempo para:

1. **Organização:** Mover todos os arquivos de IA das aulas 33 a 47 para a pasta
   correta.
2. **Padronização:** Renomear arquivos (ex: `aula43_final.png` ->
   `Aula-43-Imagem-Logotipo-EcoOffice.png`).
3. **Página de Rosto:** Criar um documento chamado `RESUMO_IA.docx` onde listam
   as ferramentas que aprenderam a usar e um parágrafo sobre sua evolução.
4. **Verificação:** Garantir que todos os PDFs estão abrindo corretamente e que
   as imagens têm boa qualidade.

### Guia para o Professor

- **Instruções Didáticas:** Circule verificando a organização. Seja rigoroso com
  nomes de arquivos genéricos como "documento1".
- **Perguntas para a Turma:**
  - "Alguém perdeu algum arquivo importante?"
  - "Qual trabalho você mais gosta de ver no seu portfólio?"
- **Dicas de Condução:** Ajude na criação de "Capas" para cada seção do
  portfólio usando as técnicas de design aprendidas.

### Entrega

- Registrar a atividade no [Formulário da Aula 48](formulario-aula-48).
- Anexar ou informar o link do arquivo, pasta, apresentação ou evidência
  produzida.
- Manter o portfólio organizado com nome de arquivo coerente e data da aula.---
