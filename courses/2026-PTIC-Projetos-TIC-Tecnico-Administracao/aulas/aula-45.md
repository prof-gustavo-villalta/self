# Aula 45 - IA em Planilhas

**Data:** 08/09/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Utilizar a IA para criar fórmulas complexas, organizar dados e
realizar análises básicas em planilhas eletrônicas (Excel/Google Sheets).

---

## 1. Abertura: O Fim do Medo das Fórmulas (10 minutos)

### A IA como Especialista em Excel

Vocês lembram da dificuldade de decorar onde colocar o ponto e vírgula no
`PROCV` ou no `SE`? Hoje a IA funciona como um tradutor: você diz em português o
que quer que a planilha faça, e ela te entrega a fórmula pronta. O segredo não é
decorar códigos, mas saber **explicar o problema**.

### Guia para o Professor

- **Instruções Didáticas:** Demonstre o ChatGPT ou o Gemini criando uma fórmula
  complexa. Ex: "Crie uma fórmula que veja se o valor na célula A1 é maior que
  100 e escreva 'Meta Batida' em azul".
- **Perguntas para a Turma:**
  - "Vocês acham que ainda é importante saber o básico de Excel se a IA faz a
    fórmula?"
  - "E se a IA errar um parêntese na fórmula, como você descobre?"
- **Dicas de Condução:** Mostre que a IA também pode explicar o que uma fórmula
  antiga e confusa faz.

---

## 2. Teoria: Comandos para Dados (15 minutos)

### O que pedir para a IA nas planilhas?

1. **Geração de Fórmulas:** "Como faço para somar apenas os valores de 'Vendas'
   que aconteceram em 'Janeiro'?".
2. **Explicação de Erros:** "Colei minha fórmula e deu erro #N/D. O que pode
   ser?".
3. **Criação de Estrutura:** "Crie uma tabela com 5 colunas para controle de
   estoque, incluindo uma coluna de 'Status' automático".
4. **Análise de Dados:** "Tenho esses dados [COLE OS DADOS]. Quais são as 3
   tendências que você percebe?".
5. **Automação (Apps Script/VBA):** Para usuários avançados, pedir pequenos
   códigos para automatizar tarefas.

### Guia para o Professor

- **Instruções Didáticas:** Explique que a IA é ótima para lógica matemática.
  Use o prompt: "Atue como um especialista em Excel".
- **Perguntas para a Turma:**
  - "Por que é perigoso colar dados reais de uma empresa no ChatGPT?"
    (Privacidade).
  - "Como a IA pode ajudar a escolher o melhor gráfico para os dados?"
- **Dicas de Condução:** Mostre as ferramentas integradas como o "Help me
  organize" do Google Sheets.

---

## 3. Atividade Prática

Os alunos devem abrir uma planilha em branco e realizar as seguintes tarefas com
suporte da IA:

1. **Tarefa 1:** Peça para a IA gerar uma lista de 10 produtos fictícios com
   preços e quantidades. Cole na planilha.
2. **Tarefa 2:** Peça uma fórmula que calcule o valor total em estoque (Preço \*
   Quantidade) para cada linha.
3. **Tarefa 3:** Peça uma fórmula de `SE` que escreva "Repor Estoque" se a
   quantidade for menor que 5.
4. **Tarefa 4 (Desafio):** Peça para a IA criar uma fórmula que conte quantos
   produtos na sua lista custam mais de R$ 50,00.

### Guia para o Professor

- **Instruções Didáticas:** Ajude os alunos a copiarem e colarem as fórmulas
  corretamente. Muitas vezes a IA usa vírgula onde o Excel brasileiro usa ponto
  e vírgula. Ensine-os a pedir: "Ajuste a fórmula para o Excel em Português
  Brasil".
- **Perguntas para a Turma:**
  - "A fórmula funcionou de primeira?"
  - "Foi mais fácil do que tentar lembrar a sintaxe sozinho?"
- **Dicas de Condução:** Peça para os alunos formatarem a planilha (cores,
  bordas) para ficar profissional.

## 5. Dicas de Ouro para o Professor

1. **Sintaxe Regional:** Este é o maior erro das IAs em planilhas. Reforce que o
   aluno deve especificar o idioma do software.
2. **Análise Crítica:** Se a IA disser que a média é 10, mas os olhos do aluno
   veem que os números são todos maiores que 100, a IA errou. Ensine-os a
   desconfiar.
3. **Limpeza de Dados:** Mostre como usar a IA para "limpar" nomes (ex: remover
   espaços extras ou transformar tudo em maiúsculas).
