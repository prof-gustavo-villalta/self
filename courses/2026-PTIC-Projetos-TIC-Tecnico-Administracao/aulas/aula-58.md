# Aula 58 - Projeto: Desenvolvimento

**Data:** 26/10/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Iniciar a construção da interface e dos fluxos principais da
solução, utilizando ferramentas de produtividade (Word, Excel, Canva, Google
Workspace).

---

## 1. Abertura: Do Desenho para a Realidade (11 minutos)

### A Hora da Verdade!

Agora que temos o cronograma e o fluxograma, vamos começar a "sujar as mãos".
Hoje é o dia de criar as planilhas, os formulários ou os documentos que o
usuário final vai ver. Foquem na funcionalidade: a solução precisa resolver o
problema que vocês definiram.

### Guia para o Professor

- **Instruções Didáticas:** Peça para os grupos abrirem o fluxograma da aula
  anterior. Ele deve ser o guia para a construção de hoje.
- **Perguntas para a Turma:**
  - "Qual ferramenta vocês escolheram para ser a interface principal?"
  - "Como será o design das tabelas ou formulários?"
- **Dicas de Condução:** Deixe-os trabalhar. O papel do professor hoje é
  **Suporte Técnico**.

---

## 2. Teoria: Design e Usabilidade (17 minutos)

### O que torna uma solução "Boa"?

1. **Clareza:** O usuário sabe o que fazer logo de cara?
2. **Eficiência:** O processo é rápido ou tem muitos cliques desnecessários?
3. **Padrão Visual:** As cores e fontes estão consistentes com a imagem da
   empresa fictícia?

### Guia para o Professor

- **Instruções Didáticas:** Explique que uma planilha bagunçada ou um formulário
  confuso desestimula o uso da ferramenta.
- **Perguntas para a Turma:**
  - "Quem no grupo está cuidando do visual (Layout)?"
- **Dicas de Condução:** Use o conceito de "Design Centrado no Usuário".

---

## 3. Atividade Prática

Os grupos devem iniciar a criação dos arquivos:

1. **Estrutura de Dados:** Crie as planilhas ou bancos de dados (Excel/Sheets).
2. **Layout do Documento:** Formate os arquivos de saída (Word/Docs).
3. **Formulário de Entrada:** Se necessário, crie o formulário de coleta de
   dados (Google Forms).

**Tarefa:** Informe no formulário qual arquivo principal foi criado hoje.

### Guia para o Professor

- **Instruções Didáticas:** Ajude com fórmulas de Excel e formatações avançadas
  de documentos.
- **Perguntas para a Turma:**
  - "Onde vai entrar a IA nesse projeto?" (Embora o foco na IA seja na próxima
    aula, eles já devem planejar o 'lugar' dela).
- **Dicas de Condução:** Garanta que os arquivos estejam salvos na pasta correta
  com nomes normatizados.

- Anexar ou informar o link do arquivo, pasta, apresentação ou evidência

---

## 5. Dicas de Ouro para o Professor

1. **Gestão de Versões:** Ensine-os a não deletarem arquivos antigos. Use "v1",
   "v2" para controle.
2. **Backup Imediato:** Certifique-se de que nada ficou apenas no computador
   local (sempre na nuvem!).
3. **Motivação:** Elogie os avanços. Ver um projeto "nascendo" é muito
   gratificante para o aluno.
