**Data:** 07/12/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática / Sala
de Aula  
**Objetivo:** Proporcionar um momento de reflexão crítica sobre o aprendizado ao
longo do ano, autoavaliação e feedback do componente.

---

## 1. Roda de Conversa: "De onde viemos e para onde vamos" (14 minutos)

### Guia para o Professor:

- **Atenção:** Esta aula foi redimensionada para 50 minutos. Mantenha o ritmo
  ágil para cobrir todo o conteúdo.
- **Reflexão:** Inicie o debate com uma pergunta aberta: "O que você achava que
  era PTIC no início do ano e o que você descobriu que é agora?".
- **Conquistas:** Peça para cada aluno citar uma habilidade nova que sente que
  domina hoje (ex: Excel, Prompt Engineering, Organização).
- **Dicas de Condução:** Crie um ambiente seguro para que os alunos possam falar
  sobre as dificuldades superadas.

---

## 2. Autoavaliação e Avaliação do Componente (18 minutos)

### Guia para o Professor:

- **Formulário Final:** Peça para os alunos preencherem o formulário oficial de
  autoavaliação (ver estrutura abaixo).
- **Processo:** Explique que a autoavaliação é parte fundamental do
  desenvolvimento profissional e pessoal.

---

## 3. Fechamento do Portfólio (18 minutos)

### Guia para o Professor:

- **Revisão Final:** Momento para os alunos garantirem que todas as evidências
  (prints, arquivos, links) estão organizadas nas pastas corretas.
- **Dicas de Condução:** Ajude quem ainda tem dúvidas sobre como gerar os links
  finais de entrega.

---

## 4. Estrutura do Formulário (Google Forms)

Formulário de Autoavaliação Final:

| Ordem | Pergunta                                                   | Tipo de Resposta | Observação                             |
| :---: | :--------------------------------------------------------- | :--------------- | :------------------------------------- |
|   1   | Nome Completo                                              | Texto Curto      | Obrigatório                            |
|   2   | Qual nota você daria para o seu esforço durante o ano?     | Escala (1-10)    | 1: Mínimo / 10: Máximo                 |
|   3   | Qual das 3 competências você mais desenvolveu?             | Múltipla Escolha | [Organização / Lógica / IA Generativa] |
|   4   | Deixe um feedback sincero sobre as aulas do Prof. Gustavo. | Parágrafo        | Ajuda na melhoria contínua             |
|   5   | Deixe um recado para você mesmo no futuro profissional!    | Parágrafo        | Inspirador                             |

---

## 5. Dicas de Ouro para o Professor

1. **Feedback Sincero:** Mostre que você também está em constante aprendizado e
   que o feedback dos alunos é valioso para melhorar as aulas do próximo ano.
2. **Reconhecimento Individual:** Se possível, faça um comentário positivo
   individual para cada aluno sobre seu progresso durante o preenchimento do
   formulário.
3. **Prazos Finais:** Reforce que este é o último momento oficial para ajustes
   no portfólio.
