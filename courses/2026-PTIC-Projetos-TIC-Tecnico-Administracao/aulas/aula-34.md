# Aula 34 - IA Tradicional vs. IAGen

**Data:** 28/07/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Diferenciar a Inteligência Artificial preditiva/tradicional da
generativa, compreendendo as aplicações ideais para cada uma no setor
administrativo.

---

## 1. Abertura: Analisar ou Criar? (10 minutos)

### Duas Faces da Mesma Moeda

Nem toda IA é igual. Algumas são ótimas para prever o futuro com base em números
(IA Tradicional), enquanto outras são artistas e escritores (IA Generativa). Na
administração, precisamos saber qual "ferramenta" usar para cada problema.

### Guia para o Professor

- **Instruções Didáticas:** Use exemplos práticos: O Netflix recomendando um
  filme (IA Tradicional) vs. O ChatGPT escrevendo o roteiro de um filme (IA
  Generativa).
- **Perguntas para a Turma:**
  - "O GPS do seu celular é IA Tradicional ou Generativa?" (Resposta:
    Tradicional/Preditiva).
  - "O filtro que te deixa velho no Instagram?" (Resposta: Generativa).
- **Dicas de Condução:** Desenhe uma tabela comparativa simples no quadro.

---

## 2. Teoria: Comparativo Técnico (15 minutos)

### IA Tradicional (Discriminativa)

- **Foco:** Classificar, identificar e prever.
- **Saída:** Um número, uma categoria, um "Sim ou Não".
- **Exemplo Adm:** Score de crédito, previsão de vendas, detecção de spam.

### IA Generativa

- **Foco:** Criar, expandir e transformar.
- **Saída:** Texto, imagem, código, áudio.
- **Exemplo Adm:** Redação de e-mails, criação de logotipos, resumo de atas.

### Guia para o Professor

- **Instruções Didáticas:** Explique que a IA Tradicional "julga" os dados,
  enquanto a Generativa "sonha" novos dados com base no que aprendeu.
- **Perguntas para a Turma:**
  - "Qual delas é mais arriscada de usar em cálculos financeiros?" (Resposta:
    Generativa, pois pode alucinar números).
  - "Qual é melhor para criar ideias de marketing?" (Resposta: Generativa).
- **Dicas de Condução:** Reforce que a IA Tradicional é mais "matemática" e a
  Generativa mais "linguística".

---

## 3. Atividade Prática

Os alunos devem analisar 5 situações administrativas e decidir qual tipo de IA
usariam e por quê:

1. Prever quantos produtos serão vendidos no Natal.
2. Escrever uma carta de desculpas para um cliente insatisfeito.
3. Identificar se uma foto de um recibo é verdadeira ou falsa.
4. Criar uma apresentação de slides sobre sustentabilidade.
5. Analisar se um currículo combina com uma vaga de emprego.

**Tarefa Extra:** No ChatGPT, peça para ele explicar a diferença entre ele
(IAGen) e um sistema de previsão de bolsa de valores (IA Tradicional).

### Guia para o Professor

- **Instruções Didáticas:** Divida a turma em grupos para discutirem as
  situações. Depois, abra para debate. Não há uma única resposta "certa"
  absoluta, o foco é a justificativa.
- **Perguntas para a Turma:**
  - "Daria para usar as duas juntas em um mesmo processo?" (Ex: IA Tradicional
    identifica um problema e IAGen escreve a solução).
- **Dicas de Condução:** Circule ajudando na argumentação técnica.

## 5. Dicas de Ouro para o Professor

1. **IA Tradicional é invisível:** Lembre-os que já usamos IA Tradicional há
   anos (filtros de e-mail, sugestões de compra) sem perceber.
2. **Fronteiras Cinzas:** Explique que hoje as ferramentas estão se fundindo. O
   Excel já usa IA Tradicional para gráficos e está ganhando IAGen para
   fórmulas.
3. **Precisão:** Destaque que para dados exatos, a IA Tradicional ainda é mais
   confiável.
