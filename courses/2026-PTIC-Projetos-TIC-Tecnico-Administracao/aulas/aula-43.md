# Aula 43 - Geração de Imagens

**Data:** 31/08/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Explorar ferramentas de geração de imagens por IA (DALL-E,
Midjourney, Canva Magic Media, Adobe Firefly) para criação de materiais visuais
administrativos e de marketing.

---

## 1. Abertura: Uma Imagem Vale mais que mil Prompts (10 minutos)

### A Revolução Visual

Até agora, falamos com a IA. Hoje, vamos pedir para ela "pintar". No mundo dos
negócios, imagens são essenciais para apresentações, redes sociais e identidade
visual. Antes, você precisava ser um designer ou pagar bancos de imagens caros.
Hoje, você precisa de imaginação e um bom prompt.

### Guia para o Professor

- **Instruções Didáticas:** Mostre uma galeria de imagens incríveis geradas por
  IA. Misture fotos reais e geradas e peça para os alunos adivinharem.
- **Perguntas para a Turma:**
  - "Quais os riscos de usar uma foto de IA em um anúncio de comida, por
    exemplo?" (Propaganda enganosa).
  - "É justo com os fotógrafos e ilustradores?"
- **Dicas de Condução:** Foque no uso ético e complementar ao trabalho de
  design.

---

## 2. Teoria: Prompts Visuais (15 minutos)

### Como descrever uma imagem?

Para imagens, o prompt muda um pouco. Além do "O quê", precisamos do "Como":

1. **Sujeito:** "Um escritório moderno com luz natural".
2. **Estilo:** "Fotografia profissional", "Ilustração 3D", "Estilo Pixar",
   "Esboço a lápis".
3. **Iluminação:** "Luz de pôr do sol", "Luz neon", "Estúdio cinematográfico".
4. **Ângulo da Câmera:** "Close-up", "Grande angular", "Vista de cima (Drone)".
5. **Cores:** "Paleta de tons pastéis", "Cores vibrantes", "Preto e branco".

### Guia para o Professor

- **Instruções Didáticas:** Explique que a IA de imagem não entende "não". Se
  você pedir "Um homem sem barba", ela verá a palavra "barba" e provavelmente a
  desenhará. Use "negativos" se a ferramenta permitir.
- **Perguntas para a Turma:**
  - "Por que é difícil para a IA desenhar mãos ou textos dentro de imagens?"
    (Evolução técnica).
  - "O que é 'direitos autorais' em imagens geradas por IA?"
- **Dicas de Condução:** Mostre como o Canva (que os alunos já conhecem)
  integrou a IA de imagem.

## 3. Atividade Prática

Os alunos devem criar um conjunto de 3 imagens para uma empresa de "Entregas
Sustentáveis":

1. **Imagem 1:** O logotipo (ou conceito de logotipo) em estilo minimalista.
2. **Imagem 2:** Uma foto publicitária de uma bicicleta elétrica de entregas em
   uma rua ensolarada.
3. **Imagem 3:** Uma ilustração 3D de um cliente feliz recebendo um pacote.

**Ferramentas Sugeridas:** Canva Magic Media (Texto para Imagem), Microsoft
Designer (DALL-E) ou Adobe Firefly.

### Guia para o Professor

- **Instruções Didáticas:** Ajude na descrição dos detalhes. Se a imagem sair
  estranha, ensine a adicionar palavras-chave como "hiper-realista" ou "8k".
- **Perguntas para a Turma:**
  - "Qual ferramenta gerou o melhor resultado?"
  - "Você usaria essas imagens em um panfleto real?"
- **Dicas de Condução:** Peça para colarem os resultados no Portfólio com os
  prompts originais abaixo de cada imagem.

## 5. Dicas de Ouro para o Professor

1. **Direitos Autorais:** Explique que, atualmente, na maioria dos países,
   imagens geradas 100% por IA não podem ter direitos autorais (são de domínio
   público).
2. **Deepfakes:** Fale brevemente sobre o perigo de criar imagens de pessoas
   reais sem autorização (será aprofundado na Aula 54).
3. **Prompt Iterativo:** Mostre que mudar uma única palavra (ex: de "noite" para
   "dia") muda toda a imagem.
