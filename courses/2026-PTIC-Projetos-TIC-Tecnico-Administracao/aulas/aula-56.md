# Aula 56 - Projeto: Planejamento

**Data:** 19/10/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Elaborar o cronograma de trabalho e o escopo detalhado da solução,
definindo as tarefas de cada integrante e as ferramentas necessárias.

---

## 1. Abertura: O Mapa da Jornada (11 minutos)

### Por que Planejar?

Um projeto sem planejamento é um convite ao caos. Hoje vamos usar as técnicas
que aprendemos no início do ano para organizar nosso tempo e recursos.
Lembrem-se: temos apenas 8 semanas até a apresentação final!

### Guia para o Professor

- **Instruções Didáticas:** Mostre um exemplo de cronograma simples (pode ser
  uma tabela no Excel ou uma ferramenta visual como Trello).
- **Perguntas para a Turma:**
  - "Qual o maior desafio do seu grupo hoje?"
  - "O que faremos se alguém do grupo faltar em uma aula importante?"
- **Dicas de Condução:** Foque na **Gestão de Tempo**.

---

## 2. Teoria: Escopo e Divisão de Tarefas (17 minutos)

### Conceitos Fundamentais

1. **Escopo:** O que o projeto VAI fazer e o que ele NÃO vai fazer.
2. **Cronograma:** As datas limites para cada entrega (Marcos).
3. **Divisão de Papéis:**
   - **Líder de Processos:** Organiza as tarefas.
   - **Especialista em IA:** Cria e refina os prompts.
   - **Especialista em TIC:** Cuida das planilhas, documentos e apresentações.
   - **Qualidade:** Revisa se tudo está correto e dentro das normas.

### Guia para o Professor

- **Instruções Didáticas:** Ajude os alunos a definirem o **MVP (Mínimo Produto
  Viável)** — o que é o essencial para o projeto funcionar.
- **Perguntas para a Turma:**
  - "Se vocês tivessem que entregar o projeto hoje, o que já teriam?"
- **Dicas de Condução:** Estimule a autonomia. Eles devem decidir quem faz o
  quê.

---

## 3. Atividade Prática

Os grupos devem criar um arquivo (Word ou Excel) contendo:

1. **Objetivo Geral:** O que a solução faz.
2. **Público-Alvo:** Quem vai usar.
3. **Cronograma de Atividades:** Uma tabela com as datas das aulas e o que será
   feito em cada uma.
4. **Lista de Ferramentas:** ChatGPT, Canva, Excel, etc.

**Tarefa:** Suba esse arquivo na pasta do grupo e informe no formulário.

### Guia para o Professor

- **Instruções Didáticas:** Passe nos grupos perguntando se as tarefas estão bem
  divididas. Cuidado com o "aluno que faz tudo sozinho".
- **Perguntas para a Turma:**
  - "Como a IA pode ajudar vocês a criarem esse cronograma?" (Sugestão: Peça
    para a IA sugerir etapas para o seu tema).
- **Dicas de Condução:** O cronograma deve ser realista. Lembre que temos o
  Sábado Letivo dia 31/10.

## 5. Dicas de Ouro para o Professor

1. **Ferramentas de Apoio:** Sugira o uso de ferramentas como o Google Calendar
   para gerenciar as datas.
2. **Flexibilidade:** Explique que o planejamento pode mudar (Plano B), mas deve
   haver uma base sólida.
3. **Documentação:** Reforce que esse planejamento fará parte do relatório
   final.
