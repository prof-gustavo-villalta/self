**Data:** 01/12/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Validar o funcionamento das soluções desenvolvidas pelos grupos em
um ambiente de "feira tecnológica" interna.

---

## 1. Preparação dos "Stands" Digitais (9 minutos)

### Guia para o Professor:

- **Atenção:** Esta aula foi redimensionada para 50 minutos. Mantenha o ritmo
  ágil para cobrir todo o conteúdo.
- **Organização:** Cada grupo deve abrir seu protótipo/solução na sua estação de
  trabalho.
- **Preparação:** Garanta que todos os links, planilhas ou bots de IA estejam
  funcionando corretamente.
- **Dicas de Condução:** Peça para os grupos prepararem um "Pitch" (apresentação
  rápida) de 2 minutos para quem visitar o stand.

---

## 2. Visitação e Teste Funcional (32 minutos)

### Guia para o Professor:

- **Dinâmica:** Metade da turma fica no seu stand, enquanto a outra metade
  visita os outros grupos para testar as soluções. Depois de 30 minutos, as
  funções se invertem.
- **Mediação:** Passe por cada stand para testar a funcionalidade técnica da
  solução. Peça para os alunos mostrarem como a IA ajudou na construção.
- **Critério Funcional:** "A solução realmente funciona?".

---

## 3. Registro e Encerramento da Feira (9 minutos)

### Guia para o Professor:

- **Registro:** Peça para os alunos tirarem prints da solução "rodando" para o
  portfólio final.
- **Dicas de Condução:** Reforce o sentimento de conquista ao ver o projeto
  finalizado e funcional.

---

## 4. Estrutura do Formulário (Google Forms)

Formulário de "Feedback do Usuário" para os colegas:

| Ordem | Pergunta                                   | Tipo de Resposta | Observação                  |
| :---: | :----------------------------------------- | :--------------- | :-------------------------- |
|   1   | Nome do Testador                           | Texto Curto      | Obrigatório                 |
|   2   | Qual projeto você acabou de testar?        | Múltipla Escolha | Lista dos projetos          |
|   3   | A interface é fácil de usar?               | Escala (1-5)     | 1: Difícil / 5: Muito fácil |
|   4   | A solução cumpre o que promete?            | Múltipla Escolha | [Sim / Não / Parcialmente]  |
|   5   | Deixe um comentário positivo para o grupo! | Parágrafo        | Incentivo                   |

---

## 5. Dicas de Ouro para o Professor

1. **Troca de Experiências:** Incentive que os alunos troquem dicas de como
   superaram desafios técnicos durante o desenvolvimento.
2. **Ambiente Real:** Mostre como as feiras de tecnologia e inovação nas
   empresas seguem este mesmo formato de demonstração direta.
3. **Fotos dos Protótipos:** Garanta que todos os grupos tenham imagens de
   qualidade da sua solução funcionando para o fechamento do ano.
