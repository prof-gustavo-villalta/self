# Aula 55 - Projeto Integrador: Definição

**Data:** 13/10/2026  
**Duração:** 1 hora-aula (50 min) **Local:** Laboratório de Informática  
**Objetivo:** Definir os grupos de trabalho e escolher o tema do Projeto
Integrador, focando em problemas reais que podem ser resolvidos com tecnologias
de TIC e IA.

---

## 1. Abertura: O Grande Desafio Final (11 minutos)

### O que é o Projeto Integrador?

Chegou a hora de aplicar tudo o que aprendemos: organização de arquivos,
planilhas, apresentações, lógica de processos e Inteligência Artificial. Vocês
vão criar uma "Solução Digital" para um problema administrativo ou educacional.
Hoje vamos formar os times e escolher o "inimigo" que vamos derrotar com
tecnologia.

### Guia para o Professor

- **Instruções Didáticas:** Explique as regras do projeto: Grupos de 3 a 4
  alunos. A solução deve obrigatoriamente usar pelo menos uma ferramenta de
  Office e uma de IA.
- **Perguntas para a Turma:**
  - "Qual processo na escola você acha que é burocrático demais e poderia ser
    digital?"
  - "Que tipo de empresa você gostaria de simular um projeto?"
- **Dicas de Condução:** Deixe claro que o projeto é a nota principal do 4º
  bimestre.

---

## 2. Teoria: Brainstorming e Problematização (17 minutos)

### Como escolher um tema?

Um bom projeto nasce de uma dor real. Pense em situações como:

1. **Gestão de Eventos:** Como organizar as inscrições de uma feira técnica?
2. **Controle Financeiro:** Como ajudar um microempreendedor a não misturar
   contas pessoais com as da empresa?
3. **Educação e IA:** Como criar um assistente de estudos para os alunos da
   Etec?
4. **RH Digital:** Como automatizar a triagem de currículos para um estágio?

### Guia para o Professor

- **Instruções Didáticas:** Use a técnica do "Post-it Digital" (Padlet, Miro ou
  mesmo um documento compartilhado) para coletar ideias iniciais.
- **Perguntas para a Turma:**
  - "Esse problema é grande demais para resolvermos em 8 semanas?" (Mantenha o
    escopo realista).
- **Dicas de Condução:** Incentive temas ligados à Administração, para alinhar
  com o curso técnico.

---

## 3. Atividade Prática

1. **Formação:** Juntem-se em grupos.
2. **Definição:** Discutam 3 ideias de problemas e escolham a melhor.
3. **Justificativa:** Por que esse problema é importante? Como a IA pode ajudar?

**Tarefa:** Preencha o formulário com os nomes dos integrantes e a ideia inicial
do projeto.

### Guia para o Professor

- **Instruções Didáticas:** Atue como um "Mentor". Se um grupo escolher algo
  muito complexo, ajude a simplificar. Se for muito simples, peça um
  diferencial.
- **Perguntas para a Turma:**
  - "Quem no grupo vai cuidar da parte de IA? Quem vai cuidar das planilhas?"
- **Dicas de Condução:** Garanta que todos os alunos estejam em algum grupo até
  o final da aula.

---

## 5. Dicas de Ouro para o Professor

1. **Equilíbrio de Grupos:** Tente equilibrar alunos com diferentes afinidades
   (um bom em lógica, outro em design, outro em organização).
2. **Protagonismo:** Deixe eles escolherem o tema, mas monitore para que o tema
   tenha "substância" administrativa.
3. **Cronograma:** Mostre o calendário das próximas aulas (Planejamento,
   Modelagem, Desenvolvimento...) para que eles sintam o ritmo do projeto.
