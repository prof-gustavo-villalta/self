## 🛠️ Ferramentas e Softwares

### Produtividade

- [Google Drive](https://drive.google.com) - Armazenamento em nuvem
- [Google Docs](https://docs.google.com) - Editor de textos
- [Google Sheets](https://sheets.google.com) - Planilhas
- [Google Slides](https://slides.google.com) - Apresentações

### Lógica e Algoritmos

- [Draw.io](https://app.diagrams.net/) - Criação de fluxogramas online
- [PSeInt](http://pseint.sourceforge.net/) - Intérprete de pseudocódigo
- [VisualG](https://visualg3.com.br/) - Alternativa ao PSeInt
- [Flowgorithm](http://www.flowgorithm.org/) - Fluxogramas executáveis
- [Scratch](https://scratch.mit.edu/) - Programação em blocos

### IA Generativa - Texto

- [ChatGPT](https://chat.openai.com/) - OpenAI
- [Microsoft Copilot](https://copilot.microsoft.com/) - Microsoft
- [Google Gemini](https://gemini.google.com/) - Google
- [Claude](https://claude.ai/) - Anthropic

### IA Generativa - Imagens

- [Bing Image Creator](https://www.bing.com/create) - DALL-E 3 gratuito
- [Midjourney](https://www.midjourney.com/) - Arte de alta qualidade
- [Leonardo.AI](https://leonardo.ai) - Game assets e personagens
- [Adobe Firefly](https://firefly.adobe.com) - Integrado ao Creative Cloud
- [Canva AI](https://www.canva.com) - Design integrado

### Segurança

- [Have I Been Pwned](https://haveibeenpwned.com/) - Verificar vazamentos
- [Bitwarden](https://bitwarden.com/) - Gerenciador de senhas
- [2FA Directory](https://2fa.directory/) - Lista de sites com 2FA
