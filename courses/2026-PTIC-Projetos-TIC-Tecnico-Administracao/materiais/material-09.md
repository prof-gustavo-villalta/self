# 📝 Guia Rápido de Formatação no Google Docs

Este material foi preparado para a **Aula 10 - Edição de Textos: Formatação**.

Ele foi pensado para alunos que estão usando o **Google Docs pela primeira
vez**. Siga cada parte com calma. O objetivo é conseguir criar um documento
simples, bonito e organizado sem depender do professor o tempo todo.

---

## O que você vai fazer hoje

Ao final da aula, você deverá conseguir:

- abrir o Google Docs,
- criar um documento novo,
- renomear o arquivo,
- digitar ou colar um texto,
- formatar o título,
- formatar os parágrafos,
- compartilhar o link do arquivo.

---

## 1. Como abrir o Google Docs

### Passo a passo

1. Abra o navegador.
2. Digite: [https://docs.google.com](https://docs.google.com)
3. Faça login com sua conta Google.
4. Clique em **Em branco** para começar.

### Se aparecer algum problema

- Se pedir login, entre com sua conta Google.
- Se a página demorar, atualize.
- Se o botão **Em branco** não aparecer, procure o sinal **+**.

---

## 2. Como renomear o documento

Quando o documento abrir:

1. Clique no nome que aparece no canto superior esquerdo.
2. Troque para um nome organizado.

### Modelo sugerido

`Aula-10-Formatacao-Nome-Sobrenome`

### Exemplo

`Aula-10-Formatacao-Maria-Silva`

---

## 3. Partes principais da tela

Ao abrir o Google Docs, você verá:

- **barra de menus**: Arquivo, Editar, Inserir, Formatar...
- **barra de ferramentas**: fonte, tamanho, negrito, alinhamento...
- **área de edição**: onde o texto é escrito
- **botão Compartilhar**: usado para enviar o link do documento

### Ferramentas que você vai usar hoje

| Ferramenta               | Para que serve                                     |
| :----------------------- | :------------------------------------------------- |
| Fonte                    | muda o tipo da letra                               |
| Tamanho da fonte         | aumenta ou diminui a letra                         |
| **B**                    | coloca em negrito                                  |
| Alinhamento              | organiza o texto à esquerda, centro ou justificado |
| Espaçamento entre linhas | aumenta o espaço entre as linhas                   |
| Compartilhar             | gera o link do documento                           |

---

## 4. Como formatar o título

O título é a primeira parte que o leitor vê. Ele deve chamar atenção sem ficar
exagerado.

### Faça assim

1. Selecione o título.
2. Coloque em **negrito**.
3. Use tamanho **14** ou **16**.
4. Deixe **centralizado**.

### Exemplo de título

**Comunicado Interno**

---

## 5. Como formatar o corpo do texto

O corpo do texto deve ficar limpo, legível e padronizado.

### Configuração recomendada

- **Fonte:** Arial ou Times New Roman
- **Tamanho:** 12
- **Alinhamento:** Justificado
- **Espaçamento entre linhas:** 1,5

### Passo a passo

1. Selecione o texto.
2. Escolha a fonte.
3. Defina o tamanho 12.
4. Clique em **Justificar**.
5. Vá em **Formatar > Espaçamento entre linhas > 1,5**.

---

## 6. O que é recuo de parágrafo

Recuo é aquele pequeno espaço no começo do parágrafo.

### Como fazer corretamente

- Use a **régua** do Google Docs.
- Ajuste a primeira linha para cerca de **1,25 cm**.

### O que não fazer

- Não use várias barras de espaço.
- Não aperte tabulação sem saber o efeito.
- Não tente alinhar texto “no olho”.

---

## 7. Erros mais comuns

Evite estes erros:

- usar muitas fontes diferentes,
- misturar tamanhos sem necessidade,
- centralizar tudo,
- apertar Enter várias vezes para “arrumar” o texto,
- esquecer de compartilhar o arquivo,
- deixar o documento com nome genérico, como `Documento sem título`.

---

## 8. Checklist antes de entregar

Antes de enviar, confira:

- [ ] O documento foi renomeado.
- [ ] O título está em destaque.
- [ ] O corpo do texto está com fonte e tamanho padronizados.
- [ ] O texto está justificado.
- [ ] O espaçamento entre linhas está correto.
- [ ] O link foi compartilhado com permissão de leitor.

---

## 9. Como compartilhar o link corretamente

1. Clique em **Compartilhar**.
2. Mude o acesso para **Qualquer pessoa com o link**.
3. Escolha a opção **Leitor**.
4. Clique em **Copiar link**.
5. Cole esse link no formulário da aula.

### Atenção

Se o link estiver como **Restrito**, o professor pode não conseguir abrir seu
arquivo.

| `Ctrl + V` | Colar           |

---

## 11. Modelo de resultado esperado

Ao final da aula, seu documento deve parecer um comunicado simples e organizado:

- título destacado,
- texto limpo,
- aparência profissional,
- leitura fácil,
- link pronto para envio.
