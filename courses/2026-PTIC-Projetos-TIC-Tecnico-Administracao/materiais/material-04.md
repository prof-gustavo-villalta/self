## O que é o Google Drive?

O Google Drive é um serviço de armazenamento em nuvem que permite:

- ☁️ Armazenar arquivos online
- 🔄 Sincronizar entre dispositivos
- 👥 Compartilhar com outras pessoas
- 📝 Criar documentos colaborativos
- 🆓 15 GB gratuitos

---

## Acesso

### Pela Web:

1. Acesse: **drive.google.com**
2. Faça login com sua conta Google
3. Pronto! Você está no seu Drive

### Pelo Computador (App):

1. Baixe em: **google.com/drive/download/**
2. Instale o aplicativo
3. Faça login
4. Uma pasta "Google Drive" aparecerá no seu computador

### Pelo Celular:

1. Baixe o app "Google Drive"
2. Faça login
3. Acesse seus arquivos de qualquer lugar

## Operações Básicas

### Criar Nova Pasta

**Método 1:**

1. Clique no botão **"+ NOVO"** (canto superior esquerdo)
2. Selecione **"Pasta"**
3. Digite o nome
4. Pressione **Enter**

**Método 2:**

1. Clique com o **botão direito** em uma área vazia
2. Selecione **"Nova pasta"**
3. Digite o nome
4. Pressione **Enter**

---

### Enviar Arquivo

**Método 1 - Upload:**

1. Clique em **"+ NOVO"**
2. Selecione **"Envio de arquivo"** ou **"Envio de pasta"**
3. Escolha o arquivo no seu computador
4. Aguarde o upload

**Método 2 - Arrastar:**

1. Abra a pasta do Google Drive no seu computador (se instalado)
2. Arraste arquivos para dentro
3. Ou arraste arquivos do navegador para a página do Drive

---

### Criar Documento

1. Clique em **"+ NOVO"**
2. Escolha o tipo:
   - **Documentos Google** (tipo Word)
   - **Planilhas Google** (tipo Excel)
   - **Apresentações Google** (tipo PowerPoint)
   - **Formulários Google** (questionários)
   - **Desenhos Google** (desenhos)
3. O documento será criado e aberto automaticamente
4. Nomeie clicando em "Documento sem título" no topo

---

### Organizar Arquivos

**Mover para pasta:**

1. Clique com o **botão direito** no arquivo
2. Selecione **"Mover para"**
3. Escolha a pasta de destino
4. Clique em **"Mover"**

**Ou arrastando:**

1. Segure o arquivo com o mouse
2. Arraste até a pasta desejada
3. Solte o botão do mouse

---

## Compartilhamento

### Compartilhar Arquivo ou Pasta

**Método 1:**

1. Clique com o **botão direito** no arquivo/pasta
2. Selecione **"Compartilhar"** (ou ícone 👤)
3. Digite o e-mail da pessoa
4. Escolha a permissão:
   - **Visualizador** → só pode ver
   - **Comentarista** → pode ver e comentar
   - **Editor** → pode ver, editar e compartilhar
5. Clique em **"Enviar"**

**Método 2 - Link:**

1. Na mesma tela de compartilhamento
2. Clique em **"Copiar link"**
3. Envie o link por WhatsApp, e-mail, etc.
4. Quem tiver o link pode acessar

---

### Níveis de Permissão

| Permissão        | Pode Ver | Pode Editar      | Pode Compartilhar   |
| ---------------- | -------- | ---------------- | ------------------- |
| **Visualizador** | ✅       | ❌               | ❌                  |
| **Comentarista** | ✅       | ❌ (só comentar) | ❌                  |
| **Editor**       | ✅       | ✅               | ✅                  |
| **Proprietário** | ✅       | ✅               | ✅ (controle total) |

---

### Revogar Acesso

1. Clique com o **botão direito** no arquivo compartilhado
2. Selecione **"Compartilhar"**
3. Clique no **X** ao lado do nome da pessoa
4. Ou clique em **"Remover acesso"**

## Sincronização

### O que é sincronização?

Sincronização mantém seus arquivos iguais em todos os dispositivos:

```
[Seu PC] ←→ [Google Drive] ←→ [Seu Celular]
    ↑                              ↑
  Edita arquivo              Acesso atualizado
  automaticamente            em qualquer lugar
```

### Como funciona:

1. Você edita um arquivo no computador
2. O Google Drive detecta a alteração
3. Sincroniza automaticamente
4. No celular, você vê a versão mais recente
5. Funciona ao contrário também!

### Configurar Sincronização

**No Windows (com app instalado):**

1. Procure "Google Drive" no menu Iniciar
2. Faça login
3. Escolha as pastas para sincronizar:
   - Todas as pastas (recomendado)
   - Apenas pastas específicas
4. Pronto! Uma pasta "Google Drive" aparecerá no Explorador

**Verificando sincronização:**

- ✅ Ícone verde → sincronizado
- ⏳ Ícone azul girando → sincronizando
- ❌ Ícone vermelho → erro

## Versões de Arquivos

### Ver Histórico de Versões

**Documentos Google:**

1. Abra o documento
2. Clique em **Arquivo** → **Histórico de versões**
3. Veja todas as alterações
4. Clique em uma versão para ver
5. Clique em **"Restaurar esta versão"** se quiser voltar

**Outros arquivos:**

1. Clique com o **botão direito** no arquivo
2. Selecione **"Gerenciar versões"**
3. Veja versões anteriores
4. Baixe ou restaure

## Dicas e Truques

### Atalhos de Teclado

| Atalho        | Ação                            |
| ------------- | ------------------------------- |
| **/**         | Focar na barra de pesquisa      |
| **Shift + T** | Criar novo documento de texto   |
| **Shift + F** | Criar nova pasta                |
| **Shift + U** | Fazer upload                    |
| **Shift + Z** | Desfazer (recuperar da lixeira) |
| **G then I**  | Ir para "Compartilhados comigo" |
| **G then T**  | Ir para "Recentes"              |

### Economizar Espaço

1. **Compactar arquivos grandes** (formato .zip)
2. **Converter para Google Docs** (não conta no espaço)
3. **Limpar lixeira** regularmente
4. **Excluir versões antigas** de arquivos grandes

### Segurança

1. ✅ Use senha forte na conta Google
2. ✅ Ative verificação em duas etapas
3. ✅ Cuidado com links compartilhados
4. ✅ Revogue acessos antigos
5. ✅ Não armazene senhas no Drive

## Resolução de Problemas

### Arquivo não sincroniza

- Verifique conexão com internet
- Reinicie o app do Google Drive
- Verifique espaço disponível

### Não consigo compartilhar

- Verifique se o arquivo está no seu Drive (não em "Compartilhados comigo")
- Verifique permissões do arquivo pai (pasta)

### Esqueci a senha

- Acesse: google.com/accounts/recovery
- Siga as instruções de recuperação
